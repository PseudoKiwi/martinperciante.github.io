Non Linear Algorithms

Introduction

Not always can we assume an input-output linear relation. What can we do in this situations? What algorithms exists? In this section, some algorithms will be studied, such as K-NN and Naive Bayes.

K-NN

K-Nearest Neighbours is an algorithm based in distances calculated in an M-dimesional space where M is the amount of attributes available. To apply the model, a proximity function and a parameter K must be defined. The proximity function is used to measure the relationship between points in our mathematical space, and K is the number of closest neighbours needed to take a decision. The class prediction made over an example will be the class with maximum presence in the example subset containing the k most near neighbours using the proximity function elected.

Naive Bayes

This algorithm roots in probability and statistic methods. More specifically, in Bayes Theorem. It computes the probability of being from class Y having attributes x using the probability of x given class Y, being x the vector of attributes. It is assumed conditional independence for the algorithm, and polynomial inputs must be used. If not, gaussian distributions will be assumed. Class Y with the biggest P(Y|x) will be the prediction.
